AgriRover

My research question is:
Can a robot reduce the amount of labor required by farmer in order to make agriculture more efficient?

My project is to design and build a robot to help farmers. The idea is simple: as the robot rolls through fields, it would identify which plants need to be trimmed via a color recognition program, and then it would cut off the sick/dead parts to allow the reste of the plant to thrive.

This would be a game changer for farmers. As of now, they don't have a tool for this. So, with the AgriRover farmers would be able to optimize their time and profit.

General Protocol

1) Cut out the wooden structure of the robot with a laser printer
2) Build a frame for the robot using a laser printer
3) Connect the motor to the Raspberry Pi and the batteries and test wiring with Python
3) Assemble the wheels onto the frame
5) put together layers on the robot to organize wiring, raspberry pi, relays, motors and the batterie
6) set up the scissors so they close automatic but have a motor open them
*Please note from here on out is what I should've done if the camera were computable
7) Install a camera to the robot's head and connect to raspberry pi
8) Connect camera to computer so it can show a livestream
9) Make a program that allows it to count black squares against white background
10) refine program so it can count more general contrasting colors
Write a program so that when the robot identifies a sick area because of a significant color difference, it sends a message to move towards that sick area, and then opens the scissors repeatedly while doing small turns until that sick area disappears
